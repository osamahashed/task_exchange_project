# Task Exchange System

## وصف المشروع
نظام ويب لإدارة وتبادل المهام بين الطالب والمشرف الأكاديمي، يتيح إنشاء المهام، إسنادها، متابعتها، وتقييمها.

## المستخدمون
- الطالب
- المشرف

## المميزات
- تسجيل دخول المستخدمين
- إنشاء وتوزيع المهام
- متابعة حالة التنفيذ
- رفع المرفقات
- تنظيم العمل الأكاديمي

## التقنيات المستخدمة
- Python 3.10
- Django
- HTML / CSS
- SQLite
- Bootstrap
